## Reporting an Issue

Found a bug, incorrect calculation, or missing data? Please open a GitHub Issue so it can be tracked and fixed.

### How to open an issue

1. Go to the [Issues tab](https://github.com/Maaaaaarrk/Hiim-PD2-Resources/issues) on this repository.
2. Click **New issue**.
3. Give your issue a clear, descriptive title (e.g. `dmg-calc: crushing blow not calculating correctly vs bosses`).
4. In the description, include:
   - **What you expected** to happen
   - **What actually happened**
   - **Steps to reproduce** (e.g. the inputs you used)
   - A screenshot or URL (the damage calculator saves state in the URL — paste it in!)
5. Click **Submit new issue**.

> You can also click the **"Report an Issue"** button directly on the [Damage Calculator](https://maaaaaarrk.github.io/Hiim-PD2-Resources/dmg-calc.html) page — it will pre-fill the GitHub issue form for you.

### Tips for a good issue report

- Be specific about which page or tool the issue is on.
- For damage calculator bugs, paste the full URL from your browser — it encodes all your inputs.
- Check [existing open issues](https://github.com/Maaaaaarrk/Hiim-PD2-Resources/issues) first to avoid duplicates.
