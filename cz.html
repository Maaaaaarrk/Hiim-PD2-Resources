Setup Discord Notifications

Get automatic Discord alerts when your favorite Corrupted Zones go live — powered by GitHub Actions. Free, no server needed, no login required.

Step 1: Create Your Own Copy
  1. Go to pd2_corrupted_zone_discord
  2. Click the green "Use this template" button → "Create a new repository"
  3. Name it whatever you like and set it to Private (recommended)
Step 2: Create a Discord Webhook
  1. In your Discord server, go to Server Settings → Integrations → Webhooks
  2. Click "New Webhook", name it, and choose a channel
  3. Click "Copy Webhook URL"
Step 3: Add the Webhook to GitHub
  1. In your new repo, go to Settings → Secrets and variables → Actions
  2. Click "New repository secret"
  3. Name: DISCORD_WEBHOOK_URL — Value: paste the webhook URL
Step 4: Done!

The workflow runs automatically every 15 minutes in sync with the zone rotation. You can also trigger it manually from the Actions tab.

Customizing Alerts

Edit config.json in your repo to control which zones send alerts:

  • Specific zones: "FAVORITE_ZONES": "Chaos Sanctuary, Cow Level"
  • By tag: "TAG_ZONES": "MF, EXP"
  • Both: alerts fire when either condition matches
  • All zones: leave both fields empty (the default)

Zone data stays in sync automatically — no updates needed when a new season starts.
